15. 문자열 포멧팅

1. +를 이용한 방식

보통 가장 많이 사용하는 방법으로, 느리다는 이야기도 있지만 크게 신경 쓸 수준은 아니다.

+ 뿐만 아니라 +=를 사용할 수 도 있다.

1
2
3
4
5
string a = "ABC" + "abc";
// 결과: ABCabc
 
string a += "456";
// 결과: ABCabc456


2. String.Format을 사용한 경우

다양한 숫자 포맷을 처리하기에 유리하다.

굉장히 강력한 숫자, 통화 표시 방법들이 있다. 아래 예시에서 {0} 부분에 number 변수가 들어간다.

1
2
3
4
int number = 2;
 
string strFormat = string.Format("There are {0} apple.", number);
// 결과: There are 2 apple.

4. 문자열 보간: &

문자열을 표시하는 부분이 적고 깔끔하다.

이것을 모르는 사람이 보면 무엇인지 못알아먹을 수 있다.

int a = 3;

// 결과: I have 3 apple and 5 candy
